## Project Description
This project is a Python-based, terminal application that allows users to create, view, modify, and delete lessons stored in JSON files. It includes both default lessons and user-created lessons.

## How to Run
1. Clone the repository
2. Run the main Python file (main.py`)
3. Follow the menu instructions in the terminal

## Language & Tools
- Language: Python
- Code Formatter: Black
- Version Control: Git

- Using conditions (if / elif / else) to control program flow
- Working with loops to handle repetitive tasks
- Handling JSON files and reading/writing data with Python
- Working with dictionaries and lists (append, update, access, delete)
